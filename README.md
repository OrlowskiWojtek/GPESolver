# GPE SOLVER

GPE Solver is C++ software for solving GPE (Gross - Pitaevski Equation).
Solver is meant to be modular as it might be used many times.

## GPE 

**Time dependent Grass Pitaevski Eqaution**:
$$ i\hbar \partial_t \psi(\vb{r}, t) = \left[ \frac{-\hbar^2}{2m} + V(\vb{r}) + g \abs{\psi}^2 + \psi_{dd} \right]\psi{\vb{r}, t} $$
